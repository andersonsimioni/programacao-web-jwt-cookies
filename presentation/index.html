JWT + Cookies + MongoDB

Aplicação Web Segura com Node.js

Ubuntu 24.04 + React + Express + MongoDB

Objetivo

  • Demonstrar autenticação segura com JWT
  • Gerenciar sessões com cookies
  • Armazenar logs e tokens no MongoDB

Arquitetura da Aplicação

  • Back-end: Node.js + Express
  • Front-end: React + Vite
  • Banco de dados: MongoDB
  • Servidor: Ubuntu 24.04 + Apache

Fluxo da Autenticação

Login → Geração de JWT → Cookie → Validação

Rotas da API

MétodoRotaDescrição
POST/loginAutenticação
POST/registerCadastro de usuário
GET/meDados do usuário
GET/log-cookiesLog de cookies
GET/cookies-dbListar logs

Interface do Usuário

  • Login / Registro
  • Visualização de cookies
  • Decodificação do JWT
